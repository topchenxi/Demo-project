##定位

html

class="pos-r|pos-a|pos-f"

.pos-r 相对定位， .pos-a 绝对定位， .pos-f 固定

##浮动

左浮动
有浮动

html

<div class="clearfix">
    <div class="fl">左浮动</div>
    <div class="fr">有浮动</div>
</div>

注意：浮动的父级元素一定要用 clearfix 清除浮动，否则高度会无法撑开。
