原因：这里填写不同原因

             <div class="hint hint-feedback hint-error">
                 <span class="icon"></span>
                 <h5>
                    对不起，您的绑定审核不通过!
                 </h5>
                 <p>
                     原因：这里填写不同原因
                 </p>

                 </p>
             </div>
